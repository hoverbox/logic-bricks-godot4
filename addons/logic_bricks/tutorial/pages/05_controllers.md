# Controllers and Logic Gates

Controllers combine the results of connected sensors and decide whether the actuators should run.

- **AND:** Every connected sensor must be true.
- **OR:** At least one connected sensor must be true.
- **NAND:** Runs unless every connected sensor is true.
- **NOR:** Runs only when none of the connected sensors are true.
- **XOR:** Exactly one connected sensor must be true.

Most beginner chains use AND. Use the other gates when multiple sensors need a different logical relationship.
