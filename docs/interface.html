Interface & Organization

Menus, graph tools, side-panel data, templates, and frames.

Add menu

The Add menu includes Search, Reroute, and the currently registered bricks. Brick availability can vary by the selected node domain.

Options menu

Copy, Paste, Duplicate

Reuse selected graph content.

Save / Load Template

Store reusable logic layouts.

Export Graph Image

Export the visible graph for documentation.

Rebuild from Script

Reconstruct metadata from generated script information.

Clear Bricks

Remove the current graph content.

Side panel

The side panel contains Variables, Globals, States, and Frames. Frames support a title, description, color, and adjustable size.
